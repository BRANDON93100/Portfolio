🗄️ Projet BTS – Gestion & sécurisation de bases MySQL (OVH / WordPress / GLPI)

MySQL phpMyAdmin OVH GLPI WordPress

J’ai travaillé sur l’exploitation et la sécurisation de bases de données MySQL utilisées par des services web : un site WordPress hébergé chez OVH et un outil GLPI (tickets/inventaire). L’objectif : garantir l’accès, corriger des problèmes de connexion et appliquer de bonnes pratiques (accès, tests, sécurité).

📌 Contexte

Les applications comme WordPress et GLPI stockent leurs données dans une base MySQL (utilisateurs, contenus, tickets, équipements, historique). Une mauvaise configuration ou un problème d’accès peut bloquer un service complet. Ce projet met en avant la compréhension et la gestion de cet élément “critique”.

🎯 Objectifs

  • Comprendre le rôle de la base MySQL dans WordPress / GLPI
  • Vérifier l’existence de la base, des utilisateurs et des droits
  • Corriger une erreur de connexion (si besoin : réinitialisation mot de passe)
  • Valider le bon fonctionnement par des tests applicatifs
  • Appliquer des bonnes pratiques (accès, confidentialité, moindre privilège)

🧩 Actions réalisées

  1. Accès au panel OVH et ouverture de phpMyAdmin.
  2. Contrôle de la base : présence, tables, cohérence générale.
  3. Vérification de l’utilisateur MySQL associé (droits / permissions).
  4. Si nécessaire : réinitialisation du mot de passe MySQL.
  5. Mise à jour de la configuration côté service (ex : paramètres DB dans WordPress / GLPI).
  6. Tests : ouverture du site / connexion GLPI / création ticket / navigation.

🛡️ Bonnes pratiques appliquées

  • Principe du moindre privilège : éviter des comptes MySQL trop permissifs.
  • Identifiants robustes et non réutilisés.
  • Vérification des accès (éviter l’exposition publique inutile).
  • Tests après changement (pour éviter les pannes cachées).

✅ Résultats

Les services utilisant la base (WordPress / GLPI) fonctionnent correctement après vérification et ajustements. Les accès sont maîtrisés et les tests confirment la stabilité de la connexion base ↔ application.

🎓 Compétences BTS mobilisées

  • Base de données : exploitation d’un SGBD (MySQL) et gestion d’accès
  • Administration / diagnostic : analyse d’un incident “connexion BDD”
  • Sécurisation : bonnes pratiques sur identifiants et permissions
  • Validation : tests fonctionnels après modification
