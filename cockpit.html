📊 Supervision Serveur Linux avec Cockpit

Interface Web Monitoring Ressources Gestion Utilisateurs Port 9090 (HTTPS)

Installation et configuration de Cockpit, une interface graphique web pour serveurs Linux (Debian/RHEL). Cet outil permet de surveiller l'état du système, gérer le stockage, les réseaux et les conteneurs sans utiliser exclusivement la ligne de commande.

📌 Contexte

Pour faciliter la maintenance quotidienne et permettre à des techniciens moins expérimentés d'intervenir sur le serveur, une interface de gestion visuelle a été mise en place. Elle permet de visualiser en temps réel la charge CPU, RAM et l'activité réseau.

🎯 Objectifs

  • Déployer la solution Cockpit sur un serveur distant
  • Permettre la gestion graphique des mises à jour système
  • Surveiller les journaux système (logs) de manière ergonomique
  • Sécuriser l'accès via le pare-feu (UFW/Firewalld)

🧩 Étapes réalisées (procédure)

  1. Installation via le gestionnaire de paquets : apt install cockpit.
  2. Activation et démarrage du service : systemctl enable --now cockpit.socket.
  3. Configuration du firewall pour autoriser le port 9090.
  4. Connexion via navigateur web sur https://adresse-ip:9090.
  5. Exploration des modules (Stockage, Réseau, Terminaux intégrés).

✅ Résultats

Le serveur est maintenant monitoré graphiquement. Les alertes de performance sont visibles immédiatement et les tâches d'administration complexes (comme l'extension de volumes de stockage) sont simplifiées et sécurisées.

🎓 Compétences mobilisées

  • Supervision : Mise en place d'un outil de monitoring temps réel
  • SISR : Maintenance corrective et évolutive du serveur
  • Support : Mise à disposition d'une console d'administration simplifiée
